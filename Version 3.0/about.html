Music Research Project - Northeastern Illinois University
Developer

Ongoing research project where the study involves analyzing the efficiency between a user taking a test in different topics with a set of genres of music, as well as taking these tests without music at all. Development tasks for creating the platform are currently the following:

  • Development of a front-end for the users to see and use through vanilla HTML/CSS/Javascript
  • Creation of an API and back-end for handling the sending of data as well as storing
  • Integration of event driven programming for interactive UI
