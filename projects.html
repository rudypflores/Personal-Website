Instagram Clone

Stack Recette

Android/IOS application that takes the user's ingredients as input. The app returns a list of possible recipes that can be done with the user's ingredients! The application was developed through the use of android studio, xcode, and programming languages such as: Java, swift and XML. The project has support for google login to eventually be able to store the users previous found recipies through the app itself as well.
